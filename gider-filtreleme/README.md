
# Gider Filtreleme Uygulaması
Bu uygulama ile giriş yapmış olduğunuz giderlernizi tarihlerine ve etiketlerine göre filtreleme yapabiliniz. 
Uygulama eğer yanlış bir format ile giriş yapıldığını tespit ederse hata gösterir.

## Nasıl çalıştırılır?

```bash
python app.py

```
